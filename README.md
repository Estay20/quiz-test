# Quiz 

Мини-приложение для прохождения тестов, написанное на чистом JS (ES6), HTML и CSS.

---

## Описание проекта

Quiz — это интерактивный тестовый модуль, который позволяет:

- проходить тест из вопросов (загружаются из JSON)
- выбирать ответы (single choice)
- переходить между вопросами
- видеть таймер обратного отсчёта
- получать результат теста
- просматривать разбор ответов

---

## Функционал

- Загрузка вопросов из `data/questions.json`
- Таймер с автозавершением
- Навигация: Назад / Далее
- Сохранение выбранных ответов (в памяти)
- Подсчёт результата (баллы и процент)
- Проверка прохождения (pass threshold)
- Режим “Разбор ответов”
- Перезапуск теста

---

## Технологии

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Local JSON (fetch API)
